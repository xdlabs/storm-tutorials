# TCP Server Spout
Simple storm tutorial in which spout acts like a TCP server.

-------------------------------------------------------------

### Installation

 * Clone this repository
```
   git clone https://github.com/xdlabs/storm-tutorials.git
``` 
 * Build sources using Maven3
```
   cd  storm-tutorials/tcp-server-spout/
   mvn package
```
